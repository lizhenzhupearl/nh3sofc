# Implementation Plans

This directory contains archived implementation plans for significant features and changes to NH3SOFC.

## Naming Convention

Plans follow the format: `YYYY-MM-DD-descriptive-title.md`

Example:
- `2026-01-22-fix-adsorbate-placer-api.md`
- `2026-01-20-add-exsolution-module.md`

## Plan Structure

Each plan should include:

```markdown
# Plan: [Descriptive Title]

**Date:** YYYY-MM-DD
**Status:** Draft | Approved | Completed | Superseded

## Summary
Brief description of what this plan addresses.

## Problem / Motivation
Why this change is needed.

## Proposed Solution
Detailed implementation approach.

## Files to Modify
- `path/to/file1.py` - Description of changes
- `path/to/file2.py` - Description of changes

## Verification
How to verify the implementation is correct.

## Notes
Any additional context or decisions made.
```
